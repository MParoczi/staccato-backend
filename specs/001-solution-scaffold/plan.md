---

## Summary

Stand up the complete 9-project Staccato solution: fix all pre-existing project
reference violations, add all required NuGet packages, configure `appsettings.json`
with four strongly typed options classes, rewrite `Program.cs` with the full
middleware pipeline (CORS+credentials, rate limiting, JWT auth, Problem Details,
`BusinessExceptionMiddleware`, SignalR, AutoMapper, FluentValidation), create the
`BusinessException` abstract base class in Domain, scaffold the `NotificationHub`,
and verify the solution builds clean with all 9 projects.

---

## Phase 0: Research Findings

See [research.md](research.md) for full detail. Key decisions:

1. **Dependency violations fixed**: Domain, Repository, Api `.csproj` files corrected.
2. **FluentValidation**: `FluentValidation` → Domain + ApiModels; `FluentValidation.AspNetCore` → Application.
3. **Rate limiting**: Global `PartitionedRateLimiter` checking path prefix — no controller attributes.
4. **BusinessException**: Abstract base in Domain with `Code`, `StatusCode`, `Details` properties.
5. **NotificationHub**: Typed `Hub<INotificationClient>` at route `/hubs/notifications`.
6. **QuestPDF**: `QuestPDF.Settings.License = LicenseType.Community` set at startup.
7. **OpenAPI removed**: `Microsoft.AspNetCore.OpenApi` removed from Application.csproj.

### Program.cs Middleware Pipeline (ordered)

```
1.  QuestPDF.Settings.License = LicenseType.Community         ← before builder.Build()
2.  builder.Services extensions:
      AddOptions<JwtOptions>().BindConfiguration("Jwt").ValidateDataAnnotations().ValidateOnStart()
      AddOptions<AzureBlobOptions>().BindConfiguration("AzureBlob").ValidateDataAnnotations().ValidateOnStart()
      AddOptions<CorsOptions>().BindConfiguration("Cors").ValidateDataAnnotations().ValidateOnStart()
      AddOptions<RateLimitOptions>().BindConfiguration("RateLimit").ValidateDataAnnotations().ValidateOnStart()
      AddCors(...)          ← reads AllowedOrigins from CorsOptions; calls AllowCredentials()
      AddRateLimiter(...)   ← GlobalLimiter with path-based /auth/* partition
      AddAuthentication(JwtBearerDefaults.AuthenticationScheme)
        .AddJwtBearer(...)  ← reads JwtOptions
      AddAuthorization()
      AddSignalR()
      AddAutoMapper(...)    ← scans Application + Api assemblies for profiles
      AddFluentValidationAutoValidation()
      AddValidatorsFromAssembly(typeof(SomeApiModelsMarker).Assembly)
      AddControllers()
      AddProblemDetails()
      AddTransient<BusinessExceptionMiddleware>()  (or register as middleware directly)
      AddHttpContextAccessor()

3.  var app = builder.Build()

4.  app.UseMiddleware<BusinessExceptionMiddleware>()   ← outermost: catch domain exceptions
5.  app.UseExceptionHandler(...)                        ← catch remaining; return Problem Details
6.  app.UseHttpsRedirection()
7.  app.UseCors("StaccatoPolicy")
8.  app.UseRateLimiter()
9.  app.UseAuthentication()
10. app.UseAuthorization()
11. app.MapHub<NotificationHub>("/hubs/notifications")
12. app.MapControllers()
13. app.Run()
```

### BusinessException Class (Domain/Exceptions/)

```csharp
namespace Domain.Exceptions;

public abstract class BusinessException : Exception
{
    public string Code { get; }
    public int StatusCode { get; protected init; } = 422;
    public object? Details { get; }

    protected BusinessException(string code, string message, object? details = null)
        : base(message)
    {
        Code = code;
        Details = details;
    }
}
```

### BusinessExceptionMiddleware (Application/Middleware/)

Intercepts `BusinessException`; writes `{ code, message, details }` as `application/json`
with the exception's `StatusCode`. All other exceptions fall through to the Problem
Details handler.

## Key Implementation Notes

- **Namespace**: Every new file uses file-scoped namespace matching its folder path.
- **Primary constructors**: Use where they reduce boilerplate (e.g., middleware constructor).
- **No magic strings**: Rate limiting policy name, CORS policy name, and hub route are all defined as `private const string` in their respective setup methods.
- **Options validation**: Use `.ValidateDataAnnotations()` + `.ValidateOnStart()` so misconfiguration causes a clear startup failure — not a silent runtime null-ref.
- **QuestPDF license**: Must be called before `builder.Build()`. Placed at top of `Program.cs`.
- **OpenAPI**: `Microsoft.AspNetCore.OpenApi` package and `app.MapOpenApi()` call are both removed.
- **Tests project references**: Add `ProjectReference` entries for all 8 other projects in `Tests.csproj`.
